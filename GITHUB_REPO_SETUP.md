# GitHub Repository Setup

Instructions for adding the documentation website link to your GitHub repository page.

## Add Website Link to Repository

1. Go to your repository on GitHub
2. Click **Settings** (top right, in the repository)
3. Scroll down to **"Features"** section
4. Find **"Website"** field
5. Enter: `https://rayraycodes.github.io/rails-accessibility-testing/`
6. Click **Save**

This will add a "Website" link in the repository's About section that appears on the main repository page.
